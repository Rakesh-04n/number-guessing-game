# 🎯 Number Guessing Game (Flask)

A simple and interactive **Number Guessing Game Web Application** built using **Flask, HTML, CSS, and SQLite**.

---

## 🚀 Features

* 🔐 User Registration & Login system
* 🎮 Guess the number game (1–100)
* ⚡ Instant result without page reload (AJAX)
* 🏆 Leaderboard (Top scores)
* 📊 Dashboard (User statistics)
* ⏱ Time tracking for each game
* 💾 Data stored using SQLite database

---

## 🛠 Technologies Used

* Python (Flask)
* HTML, CSS, JavaScript
* SQLite (Database)
* SQLAlchemy (ORM)

## ▶️ How to Run the Project

1. Clone the repository:

```
git clone https://github.com/Rakesh-04n/number-guessing-game.git
```

2. Navigate to project folder:

```
cd number-guessing-game
```

3. Install dependencies:

```
pip install flask flask-sqlalchemy werkzeug
```

4. Run the app:

```
python app.py
```

5. Open browser:

```
http://127.0.0.1:5000
```
